Request Header Parser Microservice

Project Description

This microservice analyzes your HTTP request headers and returns information about your IP address, preferred language settings, and system/browser information.

Example Usage:

Make a GET request to: [base url]/api/whoami

Example Output:

{
    "ipaddress": "159.20.14.100",
    "language": "en-US,en;q=0.5",
    "software": "Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36"
}
